## Scripts disponibles

En el directorio del proyecto puedes ejecutar comandos:

- Para iniciar la aplicación en [el puerto 3000](http://localhost:3000)
  -> `npm start`

- Para preparar la aplicación para producción en la carpeta `build`
  -> `npm run build`
